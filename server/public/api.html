SwifTicket API

Welcome to the official API documentation for SwifTicket – Universal Ticketing Platform.

Authentication & Session Routes

1. Register a New User

POST /api/auth/register

{
  "username": "john_doe",
  "email": "john@example.com",
  "password": "your_password"
}

2. Login

POST /api/auth/login

{
  "email": "john@example.com",
  "password": "your_password"
}

3. Email Verification

GET /api/auth/verified
Headers: Authorization: Bearer <token>

4. Forgot Password

POST /api/auth/forgot-password

{
  "email": "john@example.com"
}

5. Reset Password

POST /api/auth/reset-password

{
  "password": "new_password",
  "token": "reset_token"
}

6. Session Verification

POST /api/auth/jwt

{
  "token": "jwt_token_here"
}

Profile Photo Uploads

7. Upload Profile Photo

POST /api/uploads/images/users/

FormData:

  • userProfilePhoto: File
  • token: JWT

8. Get Profile Photo

GET /api/uploads/images/users?token=

Returns user’s profile image.
