SwifTicket – Cross-Industry Universal Ticketing Platform

SwifTicket is a full-stack, cross-industry movie ticket booking solution built with Vue.js, Node.js, and MongoDB. It features authentication, profile management, vendor dashboards, event discovery, booking flows, and real-time payment integrations. The platform integrates actual movie data and is built on a scalable, city-based model for multi-theatre and screen operations.

Project Overview

Frontend Features

  • Vue.js and Flexbox-based responsive UI
  • Dynamic routes with Vue Router (/, /home, /account, /events, /showtime, etc.)
  • Vuex state for auth, favorites, and session
  • Smooth transitions and guarded navigation
  • My Account tab with editing, favorites, security, and role change
  • Real-time favorite system with persistence
  • Vendor Dashboard to:
    • Create theatres and assign cities
    • Create screens with custom layouts
  • Seat selection with UI previews
  • Stripe Checkout integration
  • Confirmation screen with QR code generation
  • My Tickets tab with complete ticket history

Backend & API Features

  • Modular Express.js backend
  • MongoDB schemas: City, Theatre, Screen, Seat, User, Movie, MovieTicket
  • JWT authentication and role-based authorization
  • Nodemailer for:
    • Email verification
    • Password reset
  • Image upload support (profile & movie posters)
  • RESTful and documented API
  • Stripe integration for secure payments
  • Ticket creation and transactional metadata usage

Events & Booking System

  • Over 9,800 movie records with poster mapping
  • Filtering by genre, debounced search
  • Dynamic routes for movie details
  • "Currently Showing" badge for live screenings
  • Seat layout updates in real-time
  • Complete ticket booking flow:
    1. Choose movie, date, and screen
    2. Select available seats
    3. Pay via Stripe
    4. View summary and QR code
    5. QR codes link to ticket details
  • Blocked seats are tracked in the database

Design & UX

  • Color Palette:
    • Salmon Red: rgba(233, 106, 106)
    • Golden Yellow: #FFFC94D
  • Clean card layout with minimal shadows
  • Responsive seat grid
  • Role-based navigation
  • keep-alive used for persistent layout state
  • Toast notifications, async loaders, route guards
