SwifTicket – Versatile Ticketing Platform

SwifTicket is a versatile ticket booking full-stack platform developed using Vue.js, Node.js, and MongoDB. It includes user auth, profile management, event browsing, movie discovery, vendor theatre/screen management, and the backbone for full-fledged seat booking. It is backed by a real-world database of movies and optimized for responsiveness with debounced search, genre filtering, dynamic routing, and modular backend architecture.

Project Overview

Frontend Features

  • Redesign of the entire UI with responsive, clean layout
  • Smooth navigation with transition animation and route guards
  • Complete integration of Vue Router with dynamic routes (/, /home, /account, /event, /reset-password, /showtime)
  • Vuex-based global state management for user authentication and session
  • Profile editor with image upload, name, bio, and logout support
  • Favorites system with persistent storage and toggles on all movie cards
  • Vendor Dashboard with access to add/view theatres
  • Theatre Management includes screen creation and movie assignment with showtimes
  • Confirmation prompts for screen deletion and movie removal
  • Real-time updates reflected in the vendor panel
  • Visual seat layout editor with live rendering and data fetch
  • Events tab showing “Currently Showing” tagged movies with full booking structure
  • Search by genre, filter by tags, pick movie → pick date → view screens and showtime
  • Live layout preview under each show with movie and screen details

Backend & API Features

  • Express.js modular backend with MVC folder structure
  • Mongoose-based MongoDB models: User, UserDetails, City, Movie, Screen
  • JWT-secured role-based access (User and Vendor)
  • Nodemailer integration for email verification and password reset flows
  • Profile photo uploads and film poster image mapping
  • City schema allows for multiple cities with theatres and screens as subdocuments
  • REST API documented and structured cleanly
  • Search and filter with MongoDB aggregation
  • Favorites toggle system per user
  • Screen logic includes creation, deletion, assignment, and showtime registration
  • Movie availability per day via API for screen selection
  • Partial booking system with layout metadata exposed to UI

Events & Movie System

  • 9,800+ real movie records with posters locally stored
  • Genre chip-based filtering system
  • Async debounced movie search across all routes
  • Dedicated movie detail page with route-based access
  • Favorites toggling reflected in user dashboard and persisted
  • "Currently Showing" movies visibly separated and tag-highlighted

Design & UX

  • Navigation bar includes icons and brand-based color scheme
  • "My Account" supports editable profile with picture preview
  • Vendor-only panels hidden from regular users
  • All actions offer immediate visual feedback (favorites, showtime updates, etc.)
  • Primary visual palette uses rgba(240, 128, 128, 0.75) and #ffc94d
  • Clean UI with soft shadows, rounded borders, hover effects, and visual consistency
  • Screen layout previews are dynamically sized and reactive to state
