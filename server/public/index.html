SwifTicket – Universal Ticketing Platform

SwifTicket is a full-stack universal ticket booking platform developed using Vue.js, Node.js, and MongoDB. It provides seamless user authentication, profile management, event browsing, and movie discovery features. The platform is backed by a real-world movie dataset and optimized for responsiveness with debounced search, genre filtering, and dynamic routing.

Project Overview

Frontend Features

  • Redesigned UI layout and navigation structure
  • Responsive interfaces for login, registration, dashboard, and profile
  • Dynamic routing with route guards and animated page transitions
  • Vuex-based state management for auth and session persistence
  • Multi-page structure: /, /home, /account, /event, /reset-password
  • Editable user profile with image uploads, bio, and username update
  • Favorites system for bookmarking movies and fetching favorites

Backend & API Features

  • Modular Express server for auth, users, uploads, and favorites
  • MongoDB with Mongoose schema validation
  • JWT-based authentication for secure session handling
  • Email verification and password reset via Nodemailer
  • Static file handling for profile and movie posters
  • Favorites endpoint to fetch saved movies

Events & Movies Section

  • Integrated with 9,800+ real-world movie entries
  • Posters stored locally and linked in MongoDB
  • Debounced search bar to reduce backend load
  • Genre-based multi-select filtering
  • Dynamic route rendering for individual movie pages

User Experience & Design

  • Updated UI with modern layout and logo
  • “My Account” replaced with profile icon
  • Favorites tab added under account section
  • Role-based UI rendering for User and Vendor

Tech Stack

  • Frontend: Vue.js 2, Vuex, Vue Router, Axios
  • Backend: Node.js, Express.js, Mongoose
  • Database: MongoDB Atlas / Compass
  • Testing: Postman
