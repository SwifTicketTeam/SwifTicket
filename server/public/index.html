SwifTicket – Universal Ticketing Platform

SwifTicket is a cross-industry ticket booking full-stack solution developed with Vue.js, Node.js, and MongoDB. It includes authentication, profile management, vendor dashboards, event discovery, booking flows, and real-time payment integrations. The platform integrates real-world movie data and is designed on a scalable, city-based model for multi-theatre and screen operations.

Project Overview

Frontend Features

  • Vue.js and Flexbox-based responsive UI
  • Dynamic routes (/, /home, /account, /event, /showtime, etc.) handled by Vue Router
  • Vuex-driven state management for session, auth, favorites
  • Seamless route transitions, conditional navigation, and guarded routes
  • My Account tab with profile editing, favorites, security, and role swapping
  • Real-time favorite system with local storage and server-side persistence
  • Vendor Dashboard:
    • Create theatres and assign to cities
    • Add screens with rows, columns, and seat gaps
    • Map movies to screens with MovieCard UI
  • Seat selection and deselection by users
  • Booking interface with screen-wise time slots under movie & date
  • Stripe Checkout integrated for secure payments
  • Ticket summary shown after payment with QR code generation

Backend & API Features

  • Modular controllers for Express.js backend
  • Schemas: User, UserDetails, City, Movie, Theatre, Screen, Booking
  • JWT-based authentication with role validation
  • Nodemailer for:
    • Email verification
    • Reset/forgot password
  • Profile picture and poster uploads supported
  • Well-structured RESTful API with clear naming
  • Movie search and filter by genre/title via MongoDB queries
  • Favorite tracking and real-time syncing
  • Stripe integration:
    • Checkout session init with metadata
    • Ticket confirmation via webhook/session result

Events & Booking System

  • 9,800+ real movie dataset imported
  • Local poster assets mapped to MongoDB records
  • Debounced search + multi-genre filter on /events
  • Dynamic routing to movie detail pages
  • "Currently Showing" tag for assigned movies
  • Booking flow:
    • Movie → Date → Screen & Time → Seat
    • Payment via Stripe → Confirmation with QR

Design & UX

  • Color Theme:
    • Salmon Red: rgba(240, 128, 128, 0.75)
    • Accent Yellow: #ffc94d
  • Responsive layout with Flexbox + Vue's keep-alive
  • Form validation and toast-based feedback
  • Vendor-only visibility of management panels
  • Minimalistic card-based layout with hierarchy
  • Seat layout grid responsive to real screen sizes
